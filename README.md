# 🍫 Chocolate Sales Performance Dashboard

A **Power BI dashboard project** analyzing chocolate sales performance across products, regions, and salespersons.  
The dashboard provides insights into **revenue trends, product performance, shipment distribution, and salesperson effectiveness** to support data-driven business decisions.

# 📌 Project Overview

This project analyzes chocolate sales data to uncover key business insights such as:

- Overall sales performance
- Top-performing products
- Sales distribution across regions
- Shipment analysis
- Salesperson contribution to revenue
- Product profitability insights

The dashboard enables stakeholders to quickly understand sales patterns and make better strategic decisions.

---

# 🛠 Tech Stack

- **Power BI** – Data visualization and dashboard development  
- **Power Query** – Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** – KPI calculations and measures  
- **Excel / CSV Dataset** – Source data  

---

# 📈 Key Metrics

The dashboard includes analysis of the following KPIs:

- Total Revenue
- Total Shipments
- Total Sales
- Top Performing Products
- Sales by Region
- Sales by Salesperson
- Revenue Distribution by Product Category
